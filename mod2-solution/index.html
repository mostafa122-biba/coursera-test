topic 1

A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

topic 2

A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses (). Function names can contain letters, digits, underscores.

topic 3

Function parameters are listed inside the parentheses () in the function definition. Function arguments are the values received by the function when it is invoked.
